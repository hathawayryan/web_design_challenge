Project Summary: Latitude and Weather

The goal of this project was to look at weather data from cities of around the world and compare how the weather changes relative to latitude. This was accomplished by obtaining data for over 500 cites from OpenWeatherMap API to create a dataset.

From here, the dataset, along with tools from Matplotlib, was used to plot several graphs comparing latitude to temperature, cloudiness, wind speed, and humidity. This website includes a table of the data used for the project, along with the various graphs created during the analysis and a description of the trends observed.
